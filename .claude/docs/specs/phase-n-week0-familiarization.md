## 1. Overview

**What?** Implementare una settimana di familiarizzazione (Week 0) per utenti nuovi o che tornano dopo pausa, con 6-8 sessioni RPE-based a carico ridotto per apprendere i pattern motori fondamentali.
**Why?** Il Protocollo Phoenix §0 prescrive familiarizzazione. Iniziare con carichi di allenamento completi senza preparazione motoria aumenta il rischio infortunio e riduce l'aderenza. Il modello di apprendimento motorio di Fitts & Posner (cognitivo→associativo→autonomo) richiede tempo.
**For whom?** Utenti nuovi, utenti che tornano dopo >2 settimane di inattività, utenti che cambiano tier.
**Success metric:** Ogni nuovo utente completa Week 0 prima di iniziare il programma regolare; RPE < 6 in tutte le sessioni; pattern motori appresi.

## 2. Evidenza Scientifica

### 2.1 Motor Learning (Fitts & Posner 1967)
1. **Fase Cognitiva**: L'atleta pensa attivamente a ogni movimento — alta variabilità, errori frequenti
2. **Fase Associativa**: I movimenti diventano più fluidi — meno errori, ancora attenzione consapevole
3. **Fase Autonoma**: Il movimento è automatizzato — attenzione libera per intensità/carico

Week 0 porta l'utente dalla fase cognitiva alla fase associativa per i pattern fondamentali.

### 2.2 NSCA Guidelines (Haff & Triplett 2016)
- Nuovi praticanti: 1-2 settimane di familiarizzazione con carico leggero (50-60% 1RM stimato)
- Focus sulla tecnica, non sul carico
- 6-8 sessioni totali coprendo tutti i pattern motori

### 2.3 Tradizione Russa: Bernstein (Бернштейн Н.А., 1947)

Il framework di Bernstein **precede Fitts & Posner di 20 anni** ed è più sofisticato:

#### 5 Livelli Gerarchici (vs 3 Stadi Lineari)
| Livello | Nome | Cosa Controlla | Implicazione Week 0 |
|---------|------|---------------|---------------------|
| A | Tono/Postura | Tono muscolare, equilibrio di base | **Prerequisito esplicito** — l'Occidente lo tratta come implicito |
| B | Sinergia | Coordinazione muscolare, pattern fluidi | Dove avviene la vera coordinazione |
| C | Spazio | Orientamento nello spazio, reaching | Percezione e adattamento ambientale |
| D | Azione | Movimenti diretti a uno scopo | Livello "consapevole" |
| E | Simbolico | Linguaggio, musica, scrittura | Non rilevante per training fisico |

I livelli operano **simultaneamente** con relazioni leading/background — NON è una pipeline lineare cognitivo→associativo→autonomo.

#### "Congelamento dei Gradi di Libertà" è ADATTIVO
- I principianti "congelano" le articolazioni per ridurre la complessità del movimento
- Questo è una **strategia necessaria**, NON un errore da correggere
- **Implicazione Week 0**: Non forzare ROM completo al giorno 1. Lasciare che il corpo rilasci i gradi di libertà naturalmente

#### "Ripetizione Senza Ripetizione"
- La pratica deve essere **variabile**, non ripetitiva identica
- Lo stesso pattern motorio a velocità/range/pause diverse costruisce programmi motori migliori delle ripetizioni identiche
- I fabbri esperti mostravano PIÙ variabilità di movimento dei novizi (non meno)
- **Implicazione Week 0**: Variare leggermente angoli, velocità e pause anche durante la familiarizzazione

#### Massimo 1-2 Cue Verbali per Esercizio
- Troppe istruzioni sovraccaricano il livello conscio (D/E) e impediscono la delega al livello sinergico (B) dove vive la coordinazione reale
- **Implicazione per l'app**: Le istruzioni esercizio in Week 0 devono essere ridotte a 1-2 cue chiave, non la lista completa

**Fonti**: Bernstein, "О построении движений" (On the Construction of Movements), 1947; Bernstein "Dexterity and Its Development"; Motor learning in Russian sports science tradition

### 2.4 Tradizione Cinese: 基本功 (Jiben Gong) — Fondamenta

#### Mesi di Fondamenta Prima del "Vero" Training
Il curriculum Shaolin dedica i **primi 3 mesi interamente** a stances, flessibilità e pattern di base. Nessuna tecnica finché il corpo non ha i prerequisiti.

#### Ma Bu (Horse Stance) — Paradigma delle Hold Statiche
Le hold statiche funzionano sia come **assessment CHE training**:
- Progressione durata prima, profondità dopo
- Gate di qualità a ogni livello: 30s → 1min → 2min → 5min
- **La qualità del respiro come indicatore pass/fail** — se il respiro si rompe, la posizione non è padroneggiata
- **Implicazione Week 0**: Aggiungere plank/wall sit con quality gates basati sul respiro, non solo sul tempo

#### 5 Livelli di Sviluppo del Tai Chi
Progressione strutturata da memorizzazione esterna → sensing interno → risposta automatica, con criteri di qualità specifici a ogni gate.

#### Volume Basso, Qualità Alta nella Fase di Apprendimento Motorio (Principio Condiviso)
Sia la tradizione russa che cinese concordano:
- **NO failure muscolare** durante la fase di apprendimento
- **3-5 ripetizioni perfette > 15 ripetizioni degradate**
- Il failure degrada il pattern motorio che si sta apprendendo
- **Implicazione Week 0**: I nostri 2 set × 10-12 reps sono corretti, ma l'RPE target 4-6 deve essere RIGIDO — se l'RPE sale, fermare il set

**Fonti**: Shaolin 基本功 curriculum; Tai Chi 5 levels; Soviet multi-lateral assessment (GTO system)

### 2.5 I 6 Pattern Motori Fondamentali
1. **Squat** (dominanza ginocchio): Bodyweight squat → Goblet squat
2. **Hinge** (dominanza anca): Good morning → Romanian deadlift
3. **Push orizzontale**: Wall push-up → Push-up
4. **Pull orizzontale**: Band row → Inverted row
5. **Push verticale**: Seated press leggero → Overhead press
6. **Pull verticale**: Band lat pulldown → Lat pulldown
7. **Core**: Dead bug → Plank → Pallof press
8. **Carry**: Farmer walk (leggero)

## 3. Struttura Week 0

### 3.1 Schedule (6 sessioni in 7-9 giorni)

| Sessione | Focus | Pattern | Esercizi | RPE Target |
|----------|-------|---------|----------|------------|
| 1 | Lower Body Basics | Squat + Hinge | BW Squat, Good Morning, Glute Bridge | 4-5 |
| 2 | Upper Body Basics | Push + Pull | Wall Push-up, Band Row, Shoulder Press leggero | 4-5 |
| 3 | Core + Carry | Core + Carry | Dead Bug, Plank, Farmer Walk leggero | 4-5 |
| 4 | Lower Integration | Squat + Hinge | Goblet Squat, RDL leggero, Walking Lunge | 5-6 |
| 5 | Upper Integration | Push + Pull | Push-up (o ginocchia), Inverted Row, OHP leggero | 5-6 |
| 6 | Full Body Assessment | Tutti | 1 esercizio per pattern, 2 set leggeri | 5-6 |

### 3.2 Parametri per Sessione
- **Set**: 2 (non 3-4 come programma regolare)
- **Reps**: 10-12 (focus tecnica, non affaticamento)
- **Carico**: Bodyweight o 50-60% del carico stimato
- **Rest**: 90-120s (più del normale — tempo per assimilare)
- **Tempo**: 3-1-2-0 (eccentrica controllata, pausa, concentrica, no pausa)
- **Cue coach**: Massimo 1-2 cue per esercizio (Bernstein: troppe istruzioni bloccano il livello sinergico)
- **Variabilità**: Leggere variazioni di angolo/velocità tra i set (Bernstein: "ripetizione senza ripetizione")
- **Stop rule**: Se RPE >6, fermare il set immediatamente — il failure degrada il pattern motorio in fase di apprendimento
- **Quality gate respiro**: Se il respiro si rompe durante hold statiche (plank, wall sit), la posizione non è padroneggiata (principio 基本功)

### 3.3 Completamento e Transizione
- Week 0 è **completata** quando tutte 6 sessioni sono finite con RPE medio ≤ 6
- Se RPE > 7 in qualsiasi sessione → ripetere quella sessione
- Dopo completamento → transizione automatica al programma regolare (Beginner)
- Coach message: "Week 0 completata! Hai appreso tutti i pattern fondamentali. Ora inizia il vero programma."

## 4. Technical Approach

### 4.1 Data Model

```dart
class Week0State {
  final bool isActive;
  final int sessionsCompleted;     // 0-6
  final List<Week0Session> sessions;
  final DateTime? startedAt;
  final DateTime? completedAt;

  bool get isCompleted => sessionsCompleted >= 6;

  /// Next session to do
  int get nextSession => sessionsCompleted + 1;
}

class Week0Session {
  final int sessionNumber;        // 1-6
  final String focus;             // 'lower_basics', 'upper_basics', etc.
  final List<Week0Exercise> exercises;
  final int sets;                 // Always 2
  final int reps;                 // 10-12
  final int restSeconds;          // 90-120
  final double? avgRpe;           // Logged after completion
  final bool passed;              // avgRpe <= 6
}

class Week0Exercise {
  final String name;
  final String pattern;           // 'squat', 'hinge', 'push', 'pull', 'core', 'carry'
  final String instructions;
  final String? imageAsset;
}
```

```dart
      _session(5, 'upper_integration', ['Push-up', 'Inverted Row', 'OHP']),

```

### 4.3 Trigger Conditions

```dart
class Week0Trigger {
  /// Should this user do Week 0?
  static bool shouldActivate({
    required bool isNewUser,
    required DateTime? lastWorkoutDate,
    required String? previousTier,
    required String currentTier,
  }) {
    // New user
    if (isNewUser) return true;

    // Returning after 2+ weeks of inactivity
    if (lastWorkoutDate != null) {
      final daysSince = DateTime.now().difference(lastWorkoutDate).inDays;
      if (daysSince > 14) return true;
    }

    // Tier change (e.g., beginner → intermediate)
    if (previousTier != null && previousTier != currentTier) return true;

    return false;
  }
}
```

### 4.4 Integration with Existing Flow

1. **Onboarding**: Dopo completamento onboarding, check `Week0Trigger.shouldActivate()`
2. Se attivo → TodayScreen mostra "Week 0: Familiarizzazione" invece del programma regolare
3. **WorkoutGenerator**: Durante Week 0, genera sessioni Week 0 invece del piano standard
4. **TrainingCard**: Mostra progresso "Sessione 3/6 — Core + Carry"
5. Al completamento → transizione automatica, coach message celebrativo
